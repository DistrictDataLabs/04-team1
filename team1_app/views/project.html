Data Source

The dataset is a corpus of news and blog posts ingested using Baleen from the period March 3, 2016, ongoing. The latest dump we have received from Ben is from May 14, 2016. Currently, the data set contains 52k+ articles in HTML format for a size of approximately 3.6 GB. The corpus is still growing at a rate of several thousand posts per day. We have converted the HTML files to txt files for NLP in python.
